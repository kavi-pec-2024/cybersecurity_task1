## Overview

This report was completed as Task-1 of the Maincrafts Technology Cybersecurity & Ethical Hacking Internship. It profiles five major modern cyber threats (2024–2025), analyzes their impact on individuals and organizations, walks through a real-world case study for each, and lists practical preventive measures.

## Threats Covered

1. **AI-Powered Phishing Attacks** — case study: Arup deepfake video-conference fraud ($25M, 2024)
2. **Ransomware-as-a-Service (RaaS)** — case study: Change Healthcare / UnitedHealth Group ransomware attack (2024)
3. **Cloud Security Misconfigurations** — case study: TalentHook Azure Blob exposure (2024)
4. **IoT Vulnerabilities** — case study: Aisuru/Kimwolf IoT botnet DDoS campaign (2024–2026)
5. **Zero-Day Exploits** — case study: MOVEit Transfer zero-day exploited by Cl0p (2023–2024)
